체스 전적 검색 사이트

핵심기능:

로그인, 구독결제 시스템

유저/닉네임 전적 검색
플랫폼별(Chess.com, Lichess 등) 프로필/전적 요약
기간 필터(최근 7일/30일/전체), 시간 컨트롤(불릿/블리츠/래피드) 필터

게임 리스트 + 한 판 상세
착수 목록(PGN) + 중요 국면 하이라이트
결과/오프닝/상대 레이팅/시간 사용량

기본 통계 대시보드
승/무/패, 레이팅 추이, 연승/연패, 시간대별 성적
오프닝별 성적(백/흑, 승률, 평균 퍼포먼스)



분석/코칭급 기능 (유료 전환 포인트)

실수/악수 자동 분류
블런더/미스/부정확 + “왜 나쁜지 한 줄 설명”
반복 실수 유형 라벨링: 퀸 떨굼, 백랭크 메이트, 핀 무시, 수비 과소평가 같은 것

국면별 성적 분해
오프닝/미들/엔드게임 승률, “유리→비김/패”로 던지는 비율

시간 관리(타임 트러블) 분석
남은 시간 대비 승률, 착수당 평균 소요 시간, 시간 부족 시 blunder 급증 구간

오프닝 레퍼토리 추천
“너는 이 구조에서 승률이 높음 → 비슷한 구조 오프닝 추천”

퍼즐/훈련 생성
내 게임에서 나온 전술 패턴만 뽑아서 “맞춤 퍼즐 세트” 생성


국형으로 특히 먹히는 기능 아이디어

한글 UI/한글 해설
분석 코멘트를 짧고 직관적으로(“이 수는 중앙 장악을 잃어요” 같은)
모바일 최적화
전적 검색은 이동 중에 많이 봄 → “요약 카드”가 중요



운영/기술 관점에서 “처음부터 넣으면 위험한 것”
실시간 분석(게임 중), 대규모 엔진 분석 일괄 처리, 무제한 PGN 업로드는
서버비/큐 관리/악용 방지가 처음부터 빡세져.
대신 초반엔:
“최근 N판만 분석”, “요청한 게임만 분석”, “하루 분석 횟수 제한”
이런 식으로 안전장치 두는 게 좋아.
